# 读图C项目 QT界面Demo
### DotuAI C Project 简称DTC。

## 项目简述
	1.使用本工的风格样式的项目有C061、C062、C071、C101。（2023.11.1）
    
## 项目编译环境
    1.本工程开发语言C++ 11 
    2.依赖Qt5.13.2， OpenCV4.10 Cuda11.0 Mysql8.0 MVS3.3.1(海康) wkhtmltox(导出报表)等库文件
    3.使用VisualStudio2017编译。
    
## 项目主体功能
    1.多相机图片拼接，导出报表、时钟、正则表达式等
